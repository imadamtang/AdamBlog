介绍

Apache Hadoop 是一种开源框架，用于高效存储和处理从 GB 级到 PB 级的大型数据集。利用 Hadoop，您可以将多台计算机组成集群以便更快地并行分析海量数据集，而不是使用一台大型计算机来存储和处理数据。

三大组件

HDFS

HDFS（Hadoop Distributed File System）分布式文件系统，负责数据存储，运行在分布式集群中的不同节点上。
